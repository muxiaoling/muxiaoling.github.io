备战规划方案

注意：在复习每一个模块时，要多总结，可以通过记录文档，写博客加深记忆。

  1. 数据结构与算法

    方案：

    • 刷完公众号【代码随想录】的算法章节
    • 做LeetCode题目（数量至少400道）
    • 看完《大话数据结构》

  2. 计算机网络

    方案：

    • 看完《图解网络》PDF
    • 刷完《计算机网络自顶向下》

  3. 操作系统

    方案：

    • 看完《操作系统概念》视频
    • 看完《图解系统》PDF

  4. MySQL

    方案：

    • 刷完《MySQL45讲》
    • 刷完《MySQL技术内幕：InnoDB存储引擎》

  5. Redis

    方案：

    • 看完《Redis设计与实现》

  6. JVM

    方案：

    • 看完《深入理解JVM虚拟机》
    • 网上找几篇总结性博客看看

  7. Java基础

    方案：

    • 先把所有基础过一遍，看黑马PDF，自己总结
    • 了解Java集合源码
    • 看《Java并发编程实战》+《Java并发编程艺术》

  8. Java框架

    方案：

    • 了解Spring基础知识，源码 —— 看博客
    • 了解SpringBoot基础知识，源码 —— 看博客

  9. Linux

    方案：

    • 系统性复习Linux基础知识
    • 了解Docker基础知识，总结

  10. 项目

方案：

  • 准备一个项目，了解技术栈，难点
  • 再准备一个项目，了解技术栈，难点
